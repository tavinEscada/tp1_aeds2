Uma Solução de Localização e Navegação Indoor para o sistema MyMobiConf
Resumo. Atualmente, várias tecnologias que utilizam o GPS (Global Positioning System) 
como solução de localização estão disponíveis para diferentes campos de aplicação. 
Entretanto, a precisão dessas tecnologias pode variar bastante, especialmente no 
interior de construções, onde podem existir inúmeros bloqueios de sinal dos satélites. 
Abordagens alternativas para esses cenários, que possuem bons resultados, costumam utilizar 
infraestrutura adicional para melhorar a precisão. A abordagem do presente trabalho, por sua 
vez, usa apenas os componentes integrados em smartphones para prover um sistema de navegação 
viável para um aplicativo de apoio à participação em eventos, o myMobiConf. A proposta implementada 
se mostrou eficiente para uma vasta gama de edifícios e, portanto, diversos tipos de evento, 
além de utilizar fusão de sensores para compensar os erros inerentes aos sistemas de navegação 
inercial.
